# 模态消息组件 (Modal)

## 1. 组件概述

**模态消息组件 (Modal)** 是一个全局组件，用于显示需要用户明确响应的关键信息、确认操作或警告。它会覆盖在页面内容之上，并阻止用户与背景内容进行交互，直到用户关闭模态框为止。

## 2. 功能描述

- **中断用户流程**：强制用户处理当前信息，适用于关键决策点。
- **信息展示**：可包含标题、描述性文本、图标和操作按钮。
- **可配置性**：支持不同类型的模态框，如确认框、警告框、信息框等。

## 3. 使用场景

- **操作确认**：如“您确定要清空所有注单吗？”
- **重要警告**：如“您的账户余额不足，请先充值。”
- **关键信息通知**：如“盘口已关闭，该选项无法投注。”

## 4. 组件属性 (Props)

| 属性名 | 类型 | 必填 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| `isOpen` | boolean | 是 | `false` | 控制模态框的显示与隐藏 |
| `title` | string | 是 | - | 模态框的标题 |
| `message` | string | 是 | - | 模态框的主体信息 |
| `type` | string | 否 | `info` | 模态框类型，可选值: `info`, `warning`, `error`, `confirm` |
| `onConfirm` | function | 否 | - | 用户点击确认按钮时的回调函数 |
| `onCancel` | function | 否 | - | 用户点击取消或关闭按钮时的回调函数 |
| `confirmText` | string | 否 | `确认` | 确认按钮的文本 |
| `cancelText` | string | 否 | `取消` | 取消按钮的文本 |
