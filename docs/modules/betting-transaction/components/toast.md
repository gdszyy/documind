# Toast 组件

## 1. 组件概述

**Toast 组件** 是一个全局组件，用于显示非侵入式的、短暂的通知消息。它通常出现在屏幕的角落（如右上角或右下角），在几秒钟后会自动消失，不会中断用户的当前操作流程。

## 2. 功能描述

- **非中断性通知**：向用户提供及时的反馈信息，而无需用户进行交互。
- **自动消失**：消息在显示一段时间后会自动淡出，避免信息堆积。
- **多种状态**：支持成功、失败、信息、警告等多种状态，通过不同的颜色和图标进行区分。

## 3. 使用场景

- **操作成功提示**：如“注单已成功添加！”
- **轻量级警告**：如“赔率已更新。”
- **系统通知**：如“已自动为您切换到串关模式。”

## 4. 组件属性 (Props)

| 属性名 | 类型 | 必填 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| `message` | string | 是 | - | Toast 显示的消息内容 |
| `type` | string | 否 | `info` | Toast 类型，可选值: `info`, `success`, `warning`, `error` |
| `duration` | number | 否 | `3000` | 消息显示的持续时间（毫秒） |
| `position` | string | 否 | `top-right` | 消息显示的位置，可选值: `top-right`, `top-left`, `bottom-right`, `bottom-left` |
